# Gradebook
## Key features:
### Track your grades:
- Log your test results
- Get a chart that shows your progress
- Average grade
- Create upcoming tests so you can prepare
- Search your past tests
- Information about your subjects
- Set your goals

### Advanced features
- Make a self-assessment after the test
- Supports both the 0-100 system and the finnish 4-10 system
- Backup your information
- Create a report of all of your tests
